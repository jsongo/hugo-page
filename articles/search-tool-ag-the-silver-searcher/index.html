强大的 search 工具 - ag

介绍了 ag 工具的安装和使用方法，包括搜索文件名、常用参数等

study/工具/IT

安装

1

brew install the_silver_searcher

使用

只搜文件名

直接使用时，ag 会检索内容。需要添加 -g 参考

1

ag -g .mp4 .

正常使用

1

ag -i foo ./bar/

常用参数

– Ignore

如 --ignore *pack*

打印前后

如 -A 3 可以打印搜索结果的 after 3 行。
-B 3 可以打印 before 的 3 行

只打文件名

-l 表示 files-with-matches，即匹配的文件列表

全词匹配

-w
