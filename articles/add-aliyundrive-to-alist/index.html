阿里云盘添加到 alist

如何将阿里云盘添加到 Alist，通过填写盘名、根文件夹 ID 和获取令牌来完成配置

inbox
Thu, Apr 03, 2025

可以参考下面这个：
阿里云盘 Open | AList 文档
一般填写下面这两个即可（上方就写个盘名）

  • 根文件夹，可以在浏览器中打开相应的目录，copy 地址中的 id 即可
  • 令牌可以从这里获取：https://alist.nn.ci/zh/tool/aliyundrive/request，安全的
    其它的都不用填，保存即可。添加完回到首页就可以看到。
